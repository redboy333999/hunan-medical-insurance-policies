# 百度统计设置指南

本指南将帮助您为湖南省医保政策和法规文件网站设置百度统计，以便统计网站访问数据。

## 什么是百度统计？

百度统计是百度推出的免费网站分析工具，可以帮助您：

- 📊 **追踪访问量** - 了解每天有多少人访问您的网站
- 🌍 **分析访问来源** - 了解访问者从哪里来（搜索引擎、社交媒体、直接访问等）
- 👥 **了解访问者** - 了解访问者的地理位置、设备类型、浏览器等
- ⏱️ **分析用户行为** - 了解访问者在网站上停留了多久、浏览了哪些页面
- 📈 **优化网站** - 根据数据分析结果优化网站内容和用户体验

2. **点击"新增网站"**
   - 在页面顶部找到"管理"菜单
   - 点击"新增网站"

   - **时区：** 选择 `(GMT+08:00) 北京时间`

4. **点击"确定"**

### 步骤3：获取统计代码

1. **进入代码获取页面**
   - 在网站列表中找到您的网站

2. **复制统计代码**
   - 代码格式类似：
   ```javascript
   var _hmt = _hmt || [];
   (function() {
     var hm = document.createElement("script");

     var s = document.getElementsByTagName("script")[0]; 
     s.parentNode.insertBefore(hm, s);
   ```

## 查看分析数据

### 主要报告

#### 1. 实时访客（Realtime）
- **位置：** 实时 → 实时访客
- **用途：** 查看当前有多少人正在访问网站
- **数据：** 活跃访客数、页面浏览量、地理位置等

#### 2. 趋势分析（Trend）
- **位置：** 实时 → 趋势分析
- **用途：** 了解访问量随时间的变化
- **数据：**
  - PV（页面浏览量）
  - UV（独立访客数）
  - IP数
  - 新老访客比例

#### 3. 来源分析（Source）
- **位置：** 实时 → 来源分析
- **用途：** 了解访问者从哪里来
- **数据：**
  - 搜索引擎（百度、搜狗、360等）
  - 外部链接
  - 直接访问
  - 社交媒体

#### 4. 受众分析（Audience）
- **位置：** 实时 → 受众分析
- **用途：** 了解访问者的特征
- **数据：**
  - 地理位置（省份、城市）
  - 设备类型（桌面、移动、平板）
  - 浏览器和操作系统
  - 语言偏好
  - 新访客与回访访客比例

#### 5. 页面分析（Page）
- **位置：** 实时 → 页面分析
- **用途：** 了解哪些页面最受欢迎
- **数据：**
  - 页面浏览量
  - 平均停留时间
  - 跳出率
  - 入口页面
  - 热门页面

## 常见问题

### Q1: 为什么看不到数据？

**A:** 可能的原因：

1. **时间太短**
   - 百度统计需要24-48小时才能显示完整数据
   - 实时数据会立即显示

2. **代码未正确添加**
   - 检查 `<head>` 标签中是否正确添加了代码
   - 检查统计ID是否正确

3. **浏览器插件拦截**
   - 某些浏览器插件会拦截统计代码
   - 尝试使用无痕/隐私模式访问

4. **网站未部署**
   - 确认网站已成功部署到GitHub Pages

### Q2: 如何排除自己的访问？

**A:** 使用过滤器排除自己的IP地址：

1. 访问 https://tongji.baidu.com/
2. 点击左侧菜单的 **"设置"**
3. 点击 **"过滤规则"**
4. 点击 **"添加规则"**
5. 填写：
   - **规则名称：** 排除我的IP
   - **过滤类型：** 排除
   - **过滤条件：** IP地址等于
   - **IP地址：** 输入您的IP地址
6. 保存

**如何获取您的IP地址：**
- 访问 https://www.ip138.com/

### Q4: 数据不准确怎么办？

**A:** 可能的原因和解决方法：

1. **检查过滤规则**
   - 确认没有意外排除流量

2. **检查代码实现**
   - 确认所有页面都添加了追踪代码

3. **检查网站访问速度**
   - 如果网站加载太慢，可能影响统计

4. **联系百度统计支持**
   - 如果问题持续，联系百度统计客服

### Q5: 如何导出数据？

**A:** 导出数据的方法：

1. **在报告中导出**
   - 打开任意报告
   - 点击右上角的 **"导出"** 按钮
   - 选择格式（Excel、CSV等）

2. **使用API导出**
   - 使用百度统计API

3. **使用第三方工具**
